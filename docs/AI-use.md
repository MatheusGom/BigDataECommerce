## Onde utilizamos IA no projeto?

- Documentação: Utilizamos IA para padronizar nossa documentação e deixar todo o nosso github mais organizado e padronizado.

- Refino de funções: A IA foi aplicada no refino de funções para otimizar a execução do projeto e facilitar nossos insights.

- Consolidação do assunto: Também utilizamos para revisar e tirar dúvidas em relação as stacks utilizadas.

- Correção de Bugs: Para facilitar a progressão do projeto e redução de erros e tempo de debug.
